# 个人常用快捷键

#### 注释
`cmd + option + / `

#### 格式化代码 
自己安装了XCFormat
`control + cmd + 1`

#### 按名字查找文件
`shift + cmd + o`

#### 正在打开的界面立刻在 Navigation中找到
`shift + cmd + j`

#### 切换.h和.m文件
`control + cmd + ↑` **或** `control + cmd + ↓`

#### 回退(前进)到上个光标
`control + cmd + ←`  `control + cmd + →`

#### 折叠(展开)代码块
`opt + cmd + ←` `opt + cmd + →`

#### 查找当前文件
`cmd + f`

# 技巧

#### 当一个对象不会被使用的时候
在前面声名关键字`__unused`

#### Debug时候打印对象属性
打断点，在控制台 `po 对象.属性`
